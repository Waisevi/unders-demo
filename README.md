# app-template

This is a frontend application template intended as a starting point for applications. It provides a pre-configured setup for consistent development experience, and maintainable architecture.

## Stack

- Next.js 15
- React 19
- TypeScript
- Zod
- React Query (TanStack Query)
- React Hook Form
- PandaCSS (via styled-system/jsx)
- Storybook 8
- Jest + Testing Library
- MSW (Mock Service Worker)
- ESLint / Prettier / SonarJS

## Getting Started

```bash
  yarn install
```
